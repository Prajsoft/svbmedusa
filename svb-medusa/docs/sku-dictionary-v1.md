# SVB Sports SKU Dictionary v1

## SKU Format

`SVB-<TYPE>-<MODEL>-<COLOR>-<PACK>`

## Token Dictionaries

### TYPE

- `CRB` = cricket ball

### MODEL

- `SWFP` = Swift Plus
- `CLB` = Club
- `BLTZ` = Blitz
- `BLTZP` = Blitz Plus

### COLOR

- `RED`
- `WHT`
- `PNK`

### PACK

- `P01`
- `P06`
- `P12`

## Example SKUs

- `SVB-CRB-SWFP-WHT-P01`
- `SVB-CRB-CLB-RED-P06`
- `SVB-CRB-BLTZP-PNK-P12`

## Rules

- Uppercase only
- Hyphen-separated
- No spaces or underscores
- Model codes must be added to this dictionary before use
